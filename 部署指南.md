# 朱小翔的乐叟部落 - 部署指南

## 快速部署到 Vercel（推荐）

### 步骤 1：创建 GitHub 仓库

1. 打开 https://github.com 并登录
2. 点击右上角 **+** → **New repository**
3. 仓库名称：`lesou-tribe`
4. 选择 **Private**，点击 **Create repository**

### 步骤 2：上传代码

1. 在仓库页面点击 **uploading an existing file**
2. 将 `dist` 文件夹内的**所有文件**（包括 index.html 和 assets 文件夹）拖入上传区域
3. 点击 **Commit changes**

### 步骤 3：部署到 Vercel

1. 打开 https://vercel.com 并用 GitHub 账号登录
2. 点击 **Add New** → **Project**
3. 选择刚创建的 `lesou-tribe` 仓库
4. Framework Preset 选 **Vite**（如需要）
5. 点击 **Deploy**

部署完成后，Vercel 会给您一个 `xxx.vercel.app` 的免费域名！

---

## 自定义域名（可选）

如果想用自己的域名（如 lesou.com）：

1. 在 Vercel 项目设置 → **Domains** 中添加您的域名
2. 在域名服务商处配置 DNS 解析到 Vercel 提供的地址

---

## 更新网站内容

修改代码后，只需将更新推送到 GitHub，Vercel 会自动重新部署。
